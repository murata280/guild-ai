# Ultra-Simple Language UI 設計書（#25）

「幼稚園レベルの言葉遣い」— ひらがな・カタカナ中心、漢字30%以下で、すべての人に伝わるUI。

---

## 1. 方針

| 原則 | 内容 |
|------|------|
| ひらがな主体 | UI ラベルはひらがな・カタカナを優先。漢字は全CJK文字の30%以下 |
| 感情的な言葉 | ランクバッジに絵文字 + 短い感情語（すごい！ / いい かんじ / これから） |
| 短い文 | ボタンは2〜4文字。長いラベルはひらがなで短く |
| 親しみのある比喩 | 通帳→おこづかい帳、管理画面→わたしのページ、知能資産→ぶんしん |

## 3. ランクバッジ（RankBadge.tsx）

| ランク | 絵文字 | ラベル |
|-------|-------|-------|
| S | 👑 | すごい！ |
| A | 😊 | いい かんじ |
| B | 🌱 | これから |

- `showLabel` prop で絵文字の後ろにラベル表示
- `large` prop で大サイズ（px-4 py-1 text-base）

## 4. おこづかい貯金箱（MoneyBox.tsx）

AtoA 収益（live-earnings）が増えるたびに発動：
1. 上からコイン（🪙）が落下（0.6s `coin-drop` animation）
2. 貯金箱SVGがシェイク（0.4s `box-shake` animation）
3. 500Hz ショートベル（`playCoinChime()`）を鳴らす

### アニメーション仕様

| 項目 | 値 |
|------|-----|
| coin-drop | 0.6s, ease-out-quart, translateY(-8px → +60px), opacity 1→0 |
| box-shake | 0.4s, rotate -4°→+4°→-2°→0° |
| 音 | 500Hz (0.3s) + 1000Hz 倍音 (0.15s, delayed 30ms) |
| reduced-motion | CSS で 0.01ms に短縮（実質スキップ） |

## 5. 漢字比率 lint（kanji-ratio）

`src/lib/__tests__/jargon-lint.test.ts` に組み込み済み。

- 対象: `src/app/**/*.tsx` + `src/components/**/*.tsx`（APIルートを除く）
- 基準: JSXテキスト・文字列リテラルの漢字比率 < 30%
- allowlist: `microcopy/`（helpText層は50%まで許容）、`ValuationSection`・`WillSignalTrigger`・`TrustPanel`・`OnboardingGuide`・`CheckoutSection`・`page.tsx`・`layout.tsx`（説明テキスト多めの既存ファイル）

---

## 6. サウンド仕様

### playCoinChime()（新規）
| パーシャル | 周波数 | ゲイン | ディレイ | 長さ |
|-----------|-------|-------|---------|------|
| 1st（コインピン） | 500Hz | 0.10 | 0ms | 0.3s |
| 2nd（倍音シマー） | 1000Hz | 0.04 | 30ms | 0.15s |

- 推定音量: ~-20 dBFS（水琴窟チャイムより控えめ）
- `COIN_CHIME_FREQUENCIES = [500, 1000]` をエクスポート（テスト用）
- ミュート時 / `window` 未定義時は即 return（SSR 安全）
