# GUILD AI 用語集（Single Source of Truth）

すべてのドキュメント・コード・UI コピーで以下の表記を統一する。

| 用語 | 表記 | 意味 |
|---|---|---|
| GUILD AI | GUILD AI | 本プロジェクト名。「ギルド」＝知能を持ち寄る人間の組合 |
| Trust Score | Trust Score | 出品者の動的信用スコア（0–1000）。品質履歴・Discord 貢献度・X 拡散実績を加重合算 |
| CCAF | CCAF (Cognitive Context Audit File) | 思考プロセスの監査ファイル。思考密度・試行・人間意思シグナルを構造化 |
| Proof of Thought | Proof of Thought | CCAF をベースにプロセスを資産化する制度 |
| Ambassador | Ambassador | Discord で他者の知能資産を拡散して報酬を得る貢献者 |
| Ambassador Meritocracy | Ambassador Meritocracy | Ambassador 貢献度に応じた売上シェア・信用ランクの功績主義 |
| Trust-Based Pricing | Trust-Based Pricing | Trust Score に応じて Floor Price と露出を担保する価格制度 |
| Floor Price | Floor Price | Trust Score 連動の最低販売価格 |
| Rank | S / A / B | 知能資産の格付け。S = 魂の登記、A = 高品質、B = 標準 |
| toA | toA (to Agent) | AI エージェント向けの市場層 |
| toB | toB | 企業向けの市場層 |
| 知能鑑定士 | 知能鑑定士 (AI Auditor) | CCAF と稼働実績を解析しランクを判定するモジュール |
| 魂の登記 | 魂の登記 | 人間の意思が宿った S ランク作品の登記 |
| 黒衣 | 黒衣 | 人間を主役に立てる AI の役割比喩 |
| 柿 | 柿 (熟成する資産) | アップデートで価値が上がる登記済み知能のメタファー |
| listing_008 | listing_008 | 既存の出品 UI 構造の参照名。`src/app/sell/` で継承 |
| Vercel 稼働実績 | Vercel 稼働実績 | デプロイ・稼働状況の客観指標 |

## ランク判定の閾値（コード/ドキュメント共通）
- **S ランク**: CCAF 思考密度スコア ≥ 80 かつ Vercel 稼働日数 ≥ 30 かつ 人間意思シグナル検出
- **A ランク**: CCAF 思考密度スコア ≥ 60 かつ Vercel 稼働日数 ≥ 7
- **B ランク**: 上記未満（最低保証ランク）

## Trust Score の重み（v1）
- 品質履歴（過去登記の平均ランク重み）: 50%
- Discord 貢献度: 30%
- X 拡散実績: 20%
