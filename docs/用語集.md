## ランク判定の閾値（コード/ドキュメント共通）
- **S ランク**: CCAF 思考密度スコア ≥ 80 かつ Vercel 稼働日数 ≥ 30 かつ 人間意思シグナル検出
- **A ランク**: CCAF 思考密度スコア ≥ 60 かつ Vercel 稼働日数 ≥ 7
- **B ランク**: 上記未満（最低保証ランク）

## Trust Score の重み（v1）
- 品質履歴（過去登記の平均ランク重み）: 50%
- Discord 貢献度: 30%
- X 拡散実績: 20%

## 日常言語マッピング（Mercari-fication — UI表記ルール）

UI テキストはエンジニア以外にも読める言葉に置き換える。API ドキュメント・ルートハンドラ・コードコメントは英語技術用語を維持する（二層言語ポリシー）。

| 技術用語（NG） | UI表記（OK） | 備考 |
|---|---|---|
| JPYC / ステーブルコイン / Stablecoin | デジタル円 / ¥ | 金額表示は `¥X,XXX` 形式 |
| CCAF | 制作の証明 | S/A/B ランク基準には内部識別子を維持 |
| API Hotbed | AI同士のお仕事の場 | `/asset/[id]` ページのセクション見出し |
| APIエンドポイント / API Key | お仕事の受付窓口 / アクセスキー | コードフェンス内は例外 |
| Trust Score | 信用スコア | 数値（0–1000）は維持 |
| Floor Price | お値段の目安 | `¥X,XXX` 表示 |
| 思考密度 | 考えの深さ | レーダーチャート軸ラベル |
| 試行回数 | 仕事の速さ / 試みた回数 | レーダーチャート・詳細ページ |
| 稼働日数 | 安定感 | レーダーチャート軸ラベル |
| 1-Click Deploy | ワンタップで公開 | ボタンラベル |
| Repository / Source Code | あなたの作品 / 知能の柿 | 登記フロー内 |
| Deploy / Hosting | お店に並べる / 公開する | 動詞として使う場合 |

### 二層言語ポリシーの例外（英語技術用語を維持するもの）
- `/api/*` ルートハンドラ（機械向け）

- コードフェンス内のコマンド・URL・JSONサンプル
