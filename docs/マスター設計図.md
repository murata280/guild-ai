## 0. 設計原則
1. **魂の登記**: 人間の意思が込もったものだけを S ランクに。AI 単独生成物は S にしない。
2. **人間を主役、AI を黒衣**: AI は人間が宣伝したくなるネタを配給する裏方。
3. **柿（熟成する資産）**: 登記された知能はアップデートとコミュニティ熱量で価値が上がる。
4. **toA-First**: UI とデータ構造は AI エージェントが解釈できる JSON-LD ベースで設計。

## 2. データモデル（src/types で定義する型と一致）

### 2.1 CCAF Document（Cognitive Context Audit File）
```ts
type CCAF = {
  intentSignals: string[];   // 人間意思シグナル（魂の登記判定に利用）
  thoughtDensity: number;    // 0-100 思考密度
  iterations: number;        // 試行回数
  authorId: string;
  createdAt: string;         // ISO8601
};
```

### 2.2 Listing（出品 = 登記）
```ts
type Listing = {
  id: string;
  ownerId: string;
  title: string;
  description: string;
  ccaf: CCAF;
  vercelUptimeDays: number;
  rank: "S" | "A" | "B";
  basePrice: number;
  floorPrice: number;          // Trust-Based Pricing 由来
  githubUrl?: string;
};
```

### 2.3 TrustScoreInput
```ts
type TrustScoreInput = {
  qualityHistory: number;   // 0-100
  discordContribution: number; // 0-100
  xAmplification: number;   // 0-100
};
```

## 3. ai-auditor 仕様
- 入力: `CCAF`, `vercelUptimeDays`
- 出力: `Rank` と `score` と `reasons[]`
- 判定:
  - **S** = `thoughtDensity ≥ 80` && `vercelUptimeDays ≥ 30` && `intentSignals.length ≥ 1`
  - **A** = `thoughtDensity ≥ 60` && `vercelUptimeDays ≥ 7`
  - **B** = それ以外
- 「魂の登記」ガード: `intentSignals` が空なら自動的に S を回避。

## 4. trust-score 仕様
- 表示: 0–1000 のスコア、ランクバッジ、内訳バー、最終更新時刻
- 計算: `0.5 * qualityHistory + 0.3 * discordContribution + 0.2 * xAmplification`（0–100）→ × 10 で 0–1000 に正規化
- Floor Price 連動: `floorPrice = basePrice * (1 + score/2000)`（最大 +50%）

## 5. discord-bridge 仕様
- 入力イベント: `share`, `endorse`, `react`, `bug-report`
- アクション別重み: share=5, endorse=3, react=1, bug-report=4
- 1日あたり 50pt まで加算、ユーザー単位でレート制限
- Trust Score の `discordContribution` を更新する Pub/Sub 出力

## 6. /app/sell 仕様（listing_008 継承）
- ステップ 1: GitHub URL 入力 → リポジトリ検証
- ステップ 2: CCAF JSON アップロード or 自動生成
- ステップ 3: AI 鑑定プレビュー（ai-auditor 呼び出し）
- ステップ 4: 価格設定（basePrice、Floor Price 自動算出）
- ステップ 5: 登記送信

## 7. UI ガイド
- カラー: `#0F0F12`（黒衣）/ `#E8C46A`（柿）/ `#F5F2EA`（紙）
- フォント: Inter / Noto Sans JP
- ランクバッジ: S=金、A=銀、B=銅
