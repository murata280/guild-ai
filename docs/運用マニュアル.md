# GUILD AI 運用マニュアル（非エンジニア向け）

このマニュアルは、プログラミングの知識がなくても GUILD AI で **知能を登記して稼ぐ** ための手順を、専門用語に脚注[^1]を付けて説明します。

[^1]: 専門用語が出てきたら、各セクションの末尾に「用語の意味」をまとめています。

---

## 0. 用意するもの

- インターネットに繋がる PC またはスマートフォン
- Discord アカウント
- GitHub アカウント[^2]（無料で作れます）
- X（旧 Twitter）アカウント（任意。拡散実績の対象になります）

[^2]: GitHub = ソースコードや成果物を保管する世界共通の倉庫。GUILD AI は「ここに何を置いたか」を信頼の根拠にします。

---

## 1. 初回セットアップ

1. ブラウザで GUILD AI のトップページを開きます。
2. 右上の「ログイン」を押し、Discord アカウントで認証します。
3. ダッシュボードに自分のプロフィール（**Trust Score**[^3]）が表示されます。最初は 0 点です。

[^3]: Trust Score = あなたの信用スコア。0〜1000 の数値で、高いほど価格と露出で得をします。

---

## 2. 知能を登記する（出品する）

1. 画面左の「**登記する**（Sell）」ボタンを押します。
2. **ステップ 1**: 自分の作品を保存している GitHub リポジトリの URL[^4] を貼り付けます。
3. **ステップ 2**: **CCAF**[^5] のファイル（JSON）をアップロードします。書き方が分からない人は「自動生成」を押してください。
4. **ステップ 3**: 「AI 鑑定プレビュー」を押すと、**知能鑑定士**[^6] が S / A / B[^7] のランクを表示します。
5. **ステップ 4**: 価格（basePrice）を入力します。**Floor Price**[^8] が自動で出ます。
6. **ステップ 5**: 「登記する」ボタンを押せば完了です。

[^4]: URL = ウェブの住所。`https://github.com/...` の形をしています。
[^5]: CCAF = Cognitive Context Audit File。「あなたがどう考えてその作品を作ったか」をログとして残すファイル形式です。
[^6]: 知能鑑定士 = AI が自動でランクを付ける機能。人間の意思が入っているかを重視します（**魂の登記**）。
[^7]: ランク = S が最高、次が A、B が最低保証。S は「魂の登記」と呼ばれ、AI 単独で作ったものはなれません。
[^8]: Floor Price = 最低販売価格。Trust Score が高いほど自動で底上げされます。

---

## 3. 自分の知能資産を売れるようにする

- ランクが付くと自動で **Marketplace** に並びます。
- 価格は `Floor Price` 以上であれば自由に設定できます。
- **toA**[^9] / **toB**[^10] / 個人 のいずれかが購入できます。

[^9]: toA = AI エージェント向け。AI 同士で知能資産がやりとりされる新しい市場層。
[^10]: toB = 企業向け。

---

## 4. 拡散して稼ぐ（Ambassador）

1. Discord の `#amplify` チャンネルで、他人の登記済み資産を共有します。
2. 共有・賛同・リアクション・バグ報告ごとに **貢献ポイント**[^11] が加算されます。
3. ポイントは Trust Score と売上シェアに変換されます。

[^11]: 貢献ポイントは 1 日 50pt まで。重み: share=5, endorse=3, react=1, bug-report=4。

---

## 5. 信用が育つしくみ（柿の比喩）

- 登記された知能は時間とアップデートで価値が上がります。柿が甘くなるイメージです。
- 売れるほど・拡散されるほど **品質履歴**[^12] が育ち、Trust Score が上がります。
- Trust Score が上がると、**Floor Price** が上がり、検索でも上位に表示されます。

[^12]: 品質履歴 = 過去の購入者からのフィードバックの平均。

---

## 6. 困ったとき

- ランクが思ったより低い: CCAF の `thoughtDensity`（思考密度）の項目を見直してください。AI 任せの内容だと S にはなりません。
- Discord 連携ができない: 「設定 → 連携」で Discord を再認証してください。
- 価格を下げたい: Floor Price 未満には設定できません。Trust Score を上げると下限も柔軟になります。

---

## 7. 用語の意味（まとめ）

- **Trust Score**: 0〜1000 の信用スコア
- **CCAF**: 思考プロセスを記録するファイル形式
- **Proof of Thought**: プロセスを資産にする制度
- **Ambassador**: 拡散貢献者
- **Floor Price**: 自動で守られる最低価格
- **toA / toB**: AI エージェント向け / 企業向け市場
- **柿**: 熟成して価値が上がる登記済み知能のメタファー
